# getWebNationSkill

## 爬取全國技能競賽官網資料

這個專案旨在從全國技能競賽官方網站爬取相關資訊。

### 功能特點

- 自動抓取最新的競賽公告
- 提取重要的競賽資訊，如日期、地點和參賽條件
- 將爬取的資料整理成結構化格式，方便後續分析

### 使用方法

1. 安裝必要的 Python 套件
2. 複製 `.env.example` 檔案並重命名為 `.env`
3. 執行主程式 `main.py`
4. 爬取的資料將保存在 `download_files` 資料夾中

### 注意事項

- 請遵守網站的使用條款和爬蟲政策
- 建議設置適當的爬取間隔，以避免對目標網站造成過大負擔
